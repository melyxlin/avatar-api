About

So..what is this API?

Avatar API is a RESTful API interface to the quantified and structured data to the Avatar: The Last Airbender series and its sequel, Avatar: The Legend of Korra. Using this API allows you to consume information on the characters, creatures, locations, and events within the ATLA and LOK universe.

How much information does this API Have?

Since this is only in production phase, the API does not have a lot of data so far. However, I plan to expand on it more in the future

Wait this API is missing stuff!

I know! Feel free to contribute to open issues on Github

What is your tech stack?

The API and Website is built together in a single Python project using the Flask Framework. The website is built using pure HTML and Materialize CSS. The data is stored on static JSON files. The API is being deployed via AWS.
