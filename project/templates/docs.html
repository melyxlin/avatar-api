Documentation

Introduction

This is the documentation for Version 1 (V1) of the Avatar API, originally released in September of 2019

Reponse Format

The default response format is JSON

Authentication

This is an open API, which means that no authentication is required to query the API for data. Therefore, there is only support for GET-ing data.

ROUTES

All routes must be prefixed with https://www.avatarapi/herokuapp.com

Characters Route
GET /characters

Returns all characters.

Characters Route (using a filter)
GET /characters/param/{value}

Returns all characters whose value matches the value of the given parameter.

Param Value Description
benders string Characters with the given bending skills are included in the response.
gender string Characters with the given gender are included in the response.
ethnicity string Characters with the given ethnicity are included in the response.
nationality string Characters with the given nationality are included in the response.
affiliation string Characters who are part of the given affiliation are included in the response.
Character Route
GET /characters/{name}

Returns a specific character given a name.

Locations Route
GET /locations

Returns all the locations.

Locations Route (using a filter)
GET /locations/param/{value}

Returns all locations whose value matches the value of the given parameter.

Param Value Description
nations string Locations in the given nation are included in the response.
Location Route
GET /locations/{name}

Returns a specific location given a name.

Bendings Route
GET /bendings

Returns all bendings.

Bending Route
GET /bendings/{name}

Returns a specific bending given a name.
