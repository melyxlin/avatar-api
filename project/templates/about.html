About

So..what is this API?

Avatar API is a RESTful API interface to the quantified and structured data to the Avatar: The Last Airbender series and its sequel, Avatar: The Legend of Korra. Using this API allows you to consume information on the characters, creatures, locations, and events within the ATLA and LOK universe.

Why did you decide to create this?

I really love the Avatar (no, not the James Cameron movies) series and the richness of the environment that it is set in. Furthermore, I also wanted to explore Python a little bit and learn more about fullstack development, particularly with building and using API's and flask. .

How much information does this API Have?

Since this is only in development phase, the API does not have a lot of data so far. So far it has API endpoints for:

  • Characters
  • Locations
  • Bendings
    • But most of the data has not yet been implmented. However, I plan to expand on it more in the future.

    Wait this API is missing stuff!

    I know! Feel free to contribute to open issues on Github.

    Where is this information coming from?

    Most of the data is gathered from the website Avatar Wiki and also by just rewatching episodes Avatar: The Last Airbenderand The Legend of Korra. The data is mostly taken from the shows and not the graphic novels.

    What is your tech stack?

    The API and Website is built together in a single Python project using the Flask Framework. The website is built using pure HTML and Materialize CSS. The data is stored on static JSON files. The API is being deployed via Heroku.
